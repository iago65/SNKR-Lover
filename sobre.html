Sobre o SNKR Lover

O SNKR Lover é uma plataforma desenvolvida como parte de uma avaliação do curso de Web I, focada em compartilhar informações sobre sneakers e a cultura sneakerhead. O objetivo principal deste projeto é aplicar conceitos de desenvolvimento web, criando uma interface simples e funcional que atenda às necessidades de um público interessado nesse universo.

Minha Missão

A missão deste projeto é, primeiramente, fornecer uma plataforma simples e informativa sobre o mundo dos sneakers, destacando o meu interesse pela cultura e pelo estilo único dos tênis. O objetivo é aplicar técnicas de desenvolvimento web, como HTML e CSS, para criar uma página funcional e de fácil navegação, como parte de uma avaliação prática no curso de Web I.

Minha Visão

A visão deste projeto é desenvolver habilidades práticas na criação de sites, com foco na experiência do usuário e na clareza das informações. A ideia é criar um recurso digital acessível, onde as pessoas possam aprender mais sobre os sneakers e, ao mesmo tempo, testar e aprimorar minhas habilidades no desenvolvimento de páginas web.

Valores do Projeto

  • Aplicação de Conhecimentos
  • Simples e Funcional
  • Estudo Contínuo
  • Apreço pela cultura Sneaker e Basquete
